# subtitle-converter

# usage
python main.py [-h] path [path ...]

Converts txt to docx and viceversa.

positional arguments:

  path        Archivo a convertir

optional arguments:

  -h, --help  show help message and exit

# Dependeces
use pip install -r requirements
or execute setup.sh

# Suportted convertions
ASCII to Docx

Docx to ASCII
